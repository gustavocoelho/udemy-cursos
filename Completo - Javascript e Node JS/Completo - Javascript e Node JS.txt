VsCode (Visual Studio Code)
---------------------------
- Instalar o Plugin do Code Runner - CTRL + ALT + N para executar o código ou node .\nome_arquivo.js
- Abrir o terminar e digitar Code .\nome_arquivo.js irá abrir o Visual Code direto no arquivo. Funciona no git bash também
	- Se não estiver instalado, tem que instalar
